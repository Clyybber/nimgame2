nimgame2
========

A simple 2D game engine for Nim language.

Requires:
---------
* [sdl2_nim](https://github.com/Vladar4/sdl2_nim) package.

Recommended compilation flags:
------------------------------
-d:release --opt:speed

Status: pre-alpha
-----------------

Related projects:
-----------------
* [ng2planetoids](https://github.com/Vladar4/ng2planetoids) - a demo game.
