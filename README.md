Nimgame 2
=========

A simple 2D game engine for Nim language.

Requires:
---------
* [sdl2_nim](https://github.com/Vladar4/sdl2_nim) package.
* Runtime libraries for:
..- SDL 2.0.5
..- SDL_gfx 1.0.1
..- SDL_image 2.0.1
..- SDL_mixer 2.0.1
..- SDL_ttf 2.0.14

Recommended compilation flags:
------------------------------
-d:release --opt:speed

Status: v0.1 alpha
------------------
